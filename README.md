# actor-kotlin
Java 动态伪代理，它使用替代接口为对象创建代理。(Java dynamic pseudo proxy, which uses the stand-in interface to create a proxy for the object.)


纯Kotlin开发，使用简单但功能强大，可用于组件化开发或插件化项目开发。
